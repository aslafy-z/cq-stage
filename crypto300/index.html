Bienvenue dans le challenge Vigenère!
Alors, le Vigenère c'est quoi? Bah c'est un système de chiffrement par substitution...
Pour faire plus clair, une lettre "non chiffrée" correspondra à une autre lettre quand le message sera chiffré.
Par exemple: a=p, b=r, c=e
Donc si on écrit un message comme celui-ci: "abac", le message chiffré sera : "prpe".
C'est bon, vous avez compris? Bref, je vous donne le flag chiffré, à vous d'en faire ce que vous voulez:
rw brpx wqv : xx_ijc_xpdq
